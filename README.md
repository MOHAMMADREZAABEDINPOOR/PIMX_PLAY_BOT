# PIMX PLAY BOT 🎮🤖

A powerful Telegram bot for searching and sharing apps/files with a clean user flow, category browsing, admin tools, and automatic file cleanup.

## Highlights ✨
- Fast search experience for users (`🔍` search flow)
- Category-based navigation (`📁` categories)
- Built-in help and guided usage (`❓` help)
- Admin panel shortcuts for user stats and user list (`📊` / `📋`)
- Channel membership check before access
- Async architecture with `python-telegram-bot`, `aiohttp`, and robust task handling
- Auto-delete temporary files after sending for safer storage management

## Tech Stack 🛠️
- Python
- python-telegram-bot
- aiohttp
- AsyncIO

## Run Locally 🚀
```bash
pip install -r requirements.txt
python main.py
```

## Project Goal 🎯
Provide a practical, fast, and user-friendly Telegram bot backend for content discovery and controlled delivery.
